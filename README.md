# Sixtle

Sixtle is a Wordle-like game where players try to guess a six-letter word. Players have 7 attempts to guess the correct word, with color-coded feedback after each guess:

- 🟩 Green: Letter is in the correct position
- 🟨 Yellow: Letter is in the word but in the wrong position
- ⬛ Gray: Letter is not in the word

## Features

- 7 attempts to guess a six-letter word
- Color-coded feedback for each guess
- Hint system (2 hints available per game)
- Dark/Light theme toggle
- Responsive design
- Timer to track your solving speed

## Technologies Used

This project is built with modern web technologies:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Development

To run this project locally:

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Building

To create a production build:

```sh
npm run build
```
